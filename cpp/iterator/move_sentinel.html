std::move_sentinel

Defined in header <iterator> 
template< std::semiregular S >
class move_sentinel;
 (since C++20)

std::move_sentinel is a sentinel adaptor used for denoting ranges together with std::move_iterator.

Template parameters

S - the type of underlying sentinel

Member functions

(C++20)
