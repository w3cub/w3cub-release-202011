TypeVisitor<R,P>
@SupportedSourceVersion(RELEASE_21) public class TypeKindVisitor14<R,P> extends TypeKindVisitor9<R,P>
A visitor of types based on their kind with default behavior appropriate for source version RELEASE_14. For types Xyz that may have more than one kind, the visitXyz methods in this class delegate to the visitXyzAsKind method corresponding to the first argument's kind. The visitXyzAsKind methods call defaultAction, passing their arguments to defaultAction's corresponding parameters.
