@SupportedSourceVersion(RELEASE_8) public class TypeKindVisitor8<R,P> extends TypeKindVisitor7<R,P>
A visitor of types based on their kind with default behavior appropriate for the RELEASE_8 source version. For types Xyz that may have more than one kind, the visitXyz methods in this class delegate to the visitXyzAsKind method corresponding to the first argument's kind. The visitXyzAsKind methods call defaultAction, passing their arguments to defaultAction's corresponding parameters.
