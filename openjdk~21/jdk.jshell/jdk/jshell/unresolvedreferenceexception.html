public class UnresolvedReferenceException extends JShellException
Exception reported on attempting to execute a RECOVERABLE_DEFINED snippet.

The stack can be queried by methods on Exception. Note that in stack trace frames representing JShell Snippets, StackTraceElement.getFileName() will return "#" followed by the Snippet id and for snippets without a method name (for example an expression) StackTraceElement.getName() will be the empty string.
