Package org.w3c.dom.html

package org.w3c.dom.html
Provides interfaces for DOM Level 2 HTML Specification. Refer to the Document Object Model (DOM) Level 2 Specification, the HTML module extends the Core API to describe objects and methods specific to HTML documents that comply with HTML 4.0 Specification.
Since:
