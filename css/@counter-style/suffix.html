suffix

The suffix descriptor of the @counter-style rule specifies content that will be appended to the marker representation.

Syntax

/* <symbol> values */
suffix: "";
suffix: ") ";
suffix: url(bullet.png);

Values

<symbol>

Specifies a <symbol> that is appended to the marker representation. It may be a <string>, <image>, or <custom-ident>.

Formal definition

Formal syntax

suffix = 
  <symbol>  

<symbol> = 
  <string>        |
  <image>         |
  <custom-ident>  

<image> = 
  <url>       |
  <gradient>  

<url> = 
  url( <string> <url-modifier>* )  |
  src( <string> <url-modifier>* )

Examples

Setting a suffix for a counter

HTML

<ul class="choices">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  <li>None of the above</li>
</ul>

CSS

@counter-style options {
  system: fixed;
  symbols: A B C D;
  suffix: ") ";
}

.choices {
  list-style: options;
}
