Moon Modeler

Moon Modeler is a database design tool for MariaDB and other databases.

It allows you to draw diagrams and generate creation and validation scripts or SQL code.

Supported platforms include:

  • MariaDB
  • MongoDB
  • Mongoose

Key features:

  • Database modeling & design
  • Visualization of JSON structures and nested types
  • Support for database specific settings
  • Three display modes: metadata, sample data or descriptions
  • Default values for newly created objects
  • Export to PDF
  • Dark and Light themes
  • SQL script generation

Freeware version

Feature limited freeware version is available for Windows and Linux.

More information

See https://www.datensen.com for more information.
