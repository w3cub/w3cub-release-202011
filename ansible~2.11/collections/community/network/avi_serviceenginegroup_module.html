community.network.avi_serviceenginegroup – Module for setup of ServiceEngineGroup Avi RESTful Object

Note

This plugin is part of the community.network collection (version 2.0.1).

To install it use: ansible-galaxy collection install community.network.

To use it in a playbook, specify: community.network.avi_serviceenginegroup.

  • Synopsis
  • Requirements
  • Parameters
  • Notes
  • Examples
  • Return Values

Synopsis

  • This module is used to configure ServiceEngineGroup object
  • more examples at https://github.com/avinetworks/devops

Requirements

The below requirements are needed on the host that executes this module.

  • avisdk
