skipWhile

function stable

Returns an Observable that skips all items emitted by the source Observable as long as a specified condition holds true, but emits all further source items as soon as the condition becomes false.

skipWhile<T>(predicate: (value: T, index: number) => boolean): MonoTypeOperatorFunction<T>

Parameters

predicate

A function to test each item emitted from the source Observable.

Returns

MonoTypeOperatorFunction<T>: An Observable that begins emitting items emitted by the source Observable when the specified predicate becomes false.
