formatRelative

Represent the date in words relative to the given base date.

Description

Represent the date in words relative to the given base date.

Distance to the base date Result
Previous 6 days last Sunday at 04:30 AM
Last day yesterday at 04:30 AM
Same day today at 04:30 AM
Next day tomorrow at 04:30 AM
Next 6 days Sunday at 04:30 AM
Other 12/31/2017

Usage

// CommonJS
var formatRelative = require('date-fns/formatRelative')
// ES 2015
import formatRelative from 'date-fns/formatRelative'
// ESM
import { formatRelative } from 'date-fns'

Syntax

formatRelative(date, baseDate, [options])

Arguments

Name Description
date

the date to format
baseDate

the date to compare with
options

an object with options.
options.locale

the locale object. See [Locale]{@link https://date-fns.org/docs/Locale}
options.weekStartsOn

the index of the first day of the week (0 - Sunday)

Returns

Description

the date in words

Exceptions

Type Description
TypeError

2 arguments required
RangeError

date must not be Invalid Date
RangeError

baseDate must not be Invalid Date
RangeError

options.weekStartsOn must be between 0 and 6
RangeError

options.locale must contain localize property
RangeError

options.locale must contain formatLong property
RangeError

options.locale must contain formatRelative property

Examples

// Represent the date of 6 days ago in words relative to the given base date. In this example, today is Wednesday
const result = formatRelative(addDays(new Date(), -6), new Date())
//=> "last Thursday at 12:45 AM"
