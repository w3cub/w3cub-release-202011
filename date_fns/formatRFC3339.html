formatRFC3339

Format the date according to the RFC 3339 standard (https://tools.ietf.org/html/rfc3339#section-5.6).

Description

Return the formatted date string in RFC 3339 format. Options may be passed to control the parts and notations of the date.

Usage

// CommonJS
var formatRFC3339 = require('date-fns/formatRFC3339')
// ES 2015
import formatRFC3339 from 'date-fns/formatRFC3339'
// ESM
import { formatRFC3339 } from 'date-fns'

Syntax

formatRFC3339(date, [options])

Arguments

Name Description
date

the original date
options

an object with options.
options.fractionDigits

number of digits after the decimal point after seconds

Returns

Description

the formatted date string

Exceptions

Type Description
TypeError

1 argument required
RangeError

date must not be Invalid Date
RangeError

options.fractionDigits must be between 0 and 3

Examples

// Represent 18 September 2019 in RFC 3339 format:
const result = formatRFC3339(new Date(2019, 8, 18, 19, 0, 52))
//=> '2019-09-18T19:00:52Z'
// Represent 18 September 2019 in RFC 3339 format, 2 digits of second fraction:
const result = formatRFC3339(new Date(2019, 8, 18, 19, 0, 52, 234), { fractionDigits: 2 })
//=> '2019-09-18T19:00:52.23Z'
// Represent 18 September 2019 in RFC 3339 format, 3 digits of second fraction
const result = formatRFC3339(new Date(2019, 8, 18, 19, 0, 52, 234), { fractionDigits: 3 })
//=> '2019-09-18T19:00:52.234Z'
