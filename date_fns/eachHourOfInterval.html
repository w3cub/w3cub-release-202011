eachHourOfInterval

Return the array of hours within the specified time interval.

Description

Return the array of hours within the specified time interval.

Usage

// CommonJS
var eachHourOfInterval = require('date-fns/eachHourOfInterval')
// ES 2015
import eachHourOfInterval from 'date-fns/eachHourOfInterval'
// ESM
import { eachHourOfInterval } from 'date-fns'

Syntax

eachHourOfInterval(interval, [options])

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}
options

an object with options.
options.step

the step to increment by. The value should be more than 1.

Returns

Description

the array with starts of hours from the hour of the interval start to the hour of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.step must be a number greater than 1
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each hour between 6 October 2014, 12:00 and 6 October 2014, 15:00
const result = eachHourOfInterval({
  start: new Date(2014, 9, 6, 12),
  end: new Date(2014, 9, 6, 15)
})
//=> [
//   Mon Oct 06 2014 12:00:00,
//   Mon Oct 06 2014 13:00:00,
//   Mon Oct 06 2014 14:00:00,
//   Mon Oct 06 2014 15:00:00
// ]
