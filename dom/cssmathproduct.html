CSSMathProduct

The CSSMathProduct interface of the CSS_Object_Model#css_typed_object_model represents the result obtained by calling add(), sub(), or toSum() on CSSNumericValue. It inherits properties and methods from its parent CSSNumericValue.

Constructor

CSSMathProduct() Experimental

Creates a new CSSMathProduct object.

Instance properties

CSSMathProduct.values

Returns a CSSNumericArray object which contains one or more CSSNumericValue objects.

Static methods

The interface may also inherit methods from its parent interface, CSSMathValue.
