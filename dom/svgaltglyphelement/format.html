SVGAltGlyphElement: format property

The SVGAltGlyphElement.format property is a string that defines the format of the given font. It has the same meaning as the 'format' property of SVGGlyphRefElement property. If the font is in one of the formats listed in CSS2 section 15.3.5, then its value is the corresponding <string> parameter of the font.

Syntax

string = myGlyph.format
myGlyph.format = string

Value

The format values listed below are taken from CSS2.

Example

myGlyph.format = "truedoc-pfr";
